# グラフクエリスキル

知識グラフ（Neo4j）に蓄積された過去データを自然言語で検索する。

## 実行コマンド

```bash
python3 .claude/skills/graph-query/scripts/run_query.py "自然言語クエリ"
```

## 対応クエリ

| ユーザーの質問 | 照会内容 |
|:---|:---|
| 「7203.T の前回レポートは？」 | 直近のレポート（スコア・判定） |
| 「繰り返し候補に上がってる銘柄は？」 | 複数回スクリーニングに出現した未購入銘柄 |
| 「7203.T のリサーチ履歴」 | 過去のリサーチチェーン（日付・サマリー） |
| 「最近の市況は？」 | 直近の市場コンテキスト（指数・日付） |
| 「AAPL の取引履歴」 | 売買記録 + 投資メモ |
| 「7203.T のメモ一覧」 | 投資メモ（取引コンテキスト内） |

## 出力

結果はMarkdown形式で表示される。データが見つからない場合はその旨を表示。
Neo4j が未接続の場合は「データが見つかりませんでした」と表示。
